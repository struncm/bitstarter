Every Owners of Residential Units (ORU) have to keep all its documents on safe place with guaranteed high avalability. ORU also needs to rule everydays necessities of house with residential units and theirs owners. ORU elects one or more managers to do it. These people should communicate with unit owners, give them important informations, gather invoices from suppliers and prepare all house maintanace activities. All these features should OwnRU Manager provide and more others.

This product should also integrate accounting agenda software support. When accountant prepare billing materials, theses documents should be stored in this software and should be available for unit owners everywhere and everytime, when they need it.

Some more advanced features should be preparing for ORU meeting, presence and voting lists for these meetings and store result of these meetings. Other feature will be recording counter of electricity or water measures directly to system. These information are user for preparing billing materials.

FAQ

What is Owners of Residential Units (ORU)?
Owners of Residential Units is a type of judical person, based on owners of residential units in one house, where residential units are placed.
What means cloud software project?
OwnRU Manager will be hosted in virtual cloud environment, to minimize cost for maintanance and availability guarantee.
